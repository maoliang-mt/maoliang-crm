🏪 门店现状

香野茶咖位于杭州萧山区瓜沥镇航坞公园东100米，是一家即将开业的茶咖新店。面积100㎡，周边依山傍水，自然风景优美，具备打造「城市微度假+茶咖」特色的天然条件。

品类洞察

茶咖赛道2025-2026年持续高速增长，杭州「茶咖+」融合店态兴起。瓜沥镇属于城镇商圈，周边茶饮竞争中等偏低（晓麟家、茶理宜世等连锁覆盖但密度不高），对新进入者友好。

核心优势

  • 🌿 自然环境差异化：航坞公园旁田间风光，天然适合「打卡+休闲」场景
  • 📍 新店红利期：美团平台给予新店30天流量加权，必须抓住
  • 🎯 KP已触达：决策链短，推进速度快

核心挑战

  • ⚠️ 位置偏乡镇，自然到店客流有限，线上引流是生命线
  • ⚠️ 新店无口碑积累，需快速破零攒评
  • ⚠️ 品类同质化风险高，需打出差异记忆点

📱 线上推广：从0到1获客

店铺位于乡镇区域，自然客流有限，线上是主战场。美团系统能帮香野茶咖解决「让周边人知道你」的核心问题。

第一步：新店专属流量包（开业前7天配置）

What：利用美团新店30天加权期，配合小程序上线+扫码点餐落地页搭建。

How：

  1. 开业前3天完成美团门店信息填充（门头照≥3张、环境照≥5张、菜单100%上架含图片描述）
  2. 设置「新客专享券」：首杯立减5元（限美团小程序领取），预算500元/月
  3. 开业当天在小程序首页挂Banner「航坞公园旁·新店开业·首杯半价」
  4. 扫码点餐桌贴设计融入航坞山景元素，增加记忆锚点

第二步：美团/点评双平台铺设

What：在美团和大众点评双端建立门店形象。

How：

  1. 点评页面标题优化：「香野茶咖丨航坞山脚下的自然系咖啡」——带地标+品类+情绪词
  2. 发布3条「商户说」短内容（拍门店+出品+环境），每条带#航坞公园 标签
  3. 邀请首批5位真实顾客写好评（邀评话术：'帮我们在点评写几句，下次来送小甜品一份'）
  4. 开业首周做一波「1元品鉴券」引流（限量50份/天），快速积累消费记录

第三步：周边3公里精准获客

What：通过会员旗舰版的「潜客营销」能力触达附近用户。

How：

  1. 开通会员旗舰版后，在「潜客营销」模块设定投放半径3km
  2. 创建潜客券「新店尝鲜·任意饮品第二杯半价」，投放时段14:00-17:00（下午茶高峰）
  3. 每周检查领券-核销转化率，低于15%则调整券面额（如改为满25减8）

📲 小程序+扫码点餐：提升体验·沉淀私域

对于100㎡的茶咖店，扫码点餐不仅是提效工具，更是线下流量→私域会员的核心通道。

扫码点餐配置方案

What：每张桌台放置扫码点餐桌贴，顾客扫码即进入小程序点单。

How（逐步落地）：

  1. 桌贴设计：融入「山野绿+咖啡棕」品牌色，印上「扫我点单·免排队」+品牌logo（翟卫婷协助上传设计稿到系统后台）
  2. 菜单分类：建议分4栏——招牌特调/经典咖啡/鲜萃茶/小食甜品，每栏不超过8款（减少决策疲劳）
  3. 点单引导设置：首屏推「店长推荐」3款（毛利最高的特调），弹窗提示「加3元升大杯」
  4. 出餐叫号：对接取餐屏，顾客手机实时查看制作进度——小店也有大牌体验感

小程序会员沉淀

What：扫码点餐过程中无感引导注册会员。

How：

  1. 点餐结算页弹出「注册会员立享本单9折」（一键授权手机号即可）
  2. 注册后自动发放「7日内回头券·满20减3」（用会员旗舰版设置）
  3. 会员首页显示积分（消费1元=1分，100分=免费升杯），培养复购习惯
  4. 每周三设为「会员日·第二杯半价」，系统自动推送通知

预期效果

  • 扫码点餐上线后，前台人效提升约30%（1人即可覆盖高峰期点单）
  • 开业首月会员注册率目标≥40%（行业平均25-30%）
  • 会员回头率目标≥35%（通过回头券+积分+会员日三重驱动）

👑 会员旗舰版：从拉新到复购的完整闭环

会员旗舰版（4,888元/年）是香野茶咖线上运营的核心引擎，独有团购会员卡+潜客营销+裂变拉新三大能力。

团购会员卡（公域→私域转化器）

What：在美团/点评上架「香野茶咖会员卡」团购商品，用户购买即成为店铺会员。

How：

  1. 设置会员卡套餐：99元享120元储值+赠1杯招牌特调（感知价值>付出成本）
  2. 卡面设计突出「航坞山野·限定会员」标签，营造稀缺感
  3. 上架美团/点评双平台，关键词优化「瓜沥咖啡」「航坞公园周边」
  4. 首月促销：前100位购卡用户额外送定制帆布袋（成本8元/个，但晒图率极高）

潜客营销（精准投放）

What：向门店周边3km内、近期搜索过「咖啡/奶茶/下午茶」的用户精准推券。

How：

  1. 投放人群：25-45岁，近7天有茶饮消费或搜索行为
  2. 投放券：「到店消费满25减8」（核销门槛低，茶咖客单刚好覆盖）
  3. 投放节奏：开业首周日投100张券（预算约800元/天），后续按ROI调整
  4. 跟踪指标：领券率>20%、核销率>15%为健康，低于则调整面额或人群

裂变拉新（老带新）

What：通过会员旗舰版「任务中心」功能设置分享裂变活动。

How：

  1. 设置「邀请好友·各得半价券」：老会员分享链接，好友首次下单双方各获一杯半价券
  2. 每月设1次「拼团日」：3人成团各享7折（适合闺蜜/同事场景）
  3. 在扫码点餐完成页底部加「分享给朋友」按钮（路径最短=转化最高）

🌿 差异化：让人记住「航坞山脚的那家咖啡」

瓜沥镇不缺奶茶店，但缺有故事、有场景、值得专程来的茶咖空间。香野茶咖的核心差异化就是「山野感」。

场景打造（零成本/低成本）

What：利用现有环境打造拍照打卡场景。

How：

  1. 门口放一块手写黑板：「今日航坞山能见度：极佳☀️ 推荐坐窗边看山」——每天更新，成本0
  2. 窗边座位命名为「看山位」，菜单上标注「看山位推荐搭配：云雾拿铁」（产品与场景绑定）
  3. 提供拍照道具：竹编托盘+绿植背景板，暗示顾客拍照发圈（免费UGC内容）
  4. 定制纸杯印「航坞山脚·香野出品」+简笔山景插画（杯子本身就是传播媒介）

产品差异化

What：打造2-3款「只有这里才有」的招牌产品。

How：

  1. 「航坞绿」特调：抹茶+椰乳+薄荷叶，取名呼应航坞山（成本约4元，定价22元）
  2. 「山野拿铁」：咖啡+桂花糖浆+燕麦奶，秋冬主推（季节感+在地食材）
  3. 小程序点单页给招牌款加🔥标+「本店独家」tag
  4. 每月限定1款「当季特调」，培养顾客定期探店动机

线上内容运营

What：通过点评/小红书内容持续放大「山野茶咖」心智。

How：

  1. 点评商户说每周更新1条（出品+环境+山景，带话题#航坞公园打卡）
  2. 制作「从航坞公园走路到香野只要5分钟」短内容，关联周边POI
  3. 邀请小红书本地博主到店（赠饮+免费出片），换1条探店笔记

开店容易守店难。方案落地后，必须建立「看数→判断→调整」的周循环。

月度复盘模板

每月1号输出上月复盘（翟卫婷协助拉取系统后台数据）：

  1. 流量端：美团曝光量/点击率/到店转化率变化趋势
  2. 转化端：新客数/新会员数/首单转化率
  3. 复购端：会员活跃率/复购频次/沉睡会员唤醒率
  4. 利润端：毛利率/活动补贴成本占比（控制在营收5%以内）

持续优化方向

  • 📈 旺季加码：春夏航坞公园游客增多，可推「徒步打卡套餐」（饮品+能量棒）
  • 🤝 异业联动：与航坞公园/周边民宿合作，互相导流（放对方优惠单页）
  • 🔄 菜单迭代：每季更新30%产品，保持新鲜感；保留数据验证过的TOP5常青款
  • 💡 外卖试水：稳定期后可考虑开通外卖（3km配送），进一步扩大覆盖
