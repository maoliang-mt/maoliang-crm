📉
新客进店率不稳定
面食赛道竞品密集，点评评分和排名直接影响自然曝光。没有稳定的点评运营，流量被强竞品截走。
→ 点评评分提升 + 好评引导是首要任务
🔄
老客复购靠"感情"不靠"机制"
快餐客群理应是高频复购，但没有会员体系锁客，老客来了也不知道，走了也找不回来。
→ 扫码入会 + 储值卡机制亟待建立
⚡
高峰期出餐效率瓶颈
面食午市高峰11:30-13:00，人工点单效率低，排队体验差，容易造成流失。出餐慢是快餐的致命伤。
→ 扫码点餐 + KDS厨显提速是核心
💰
客单价天花板低
单碗面20-35元，附加消费少，小食/饮料点单率极低。缺少有效的加购引导，坪效无法提升。
→ 套餐组合 + 加购弹窗拉升客单价
🌐
外卖占比偏低
杭州互联网人密集，外卖渗透率极高（50%+），但面食外卖在周边写字楼/小区的触达还有巨大空间。
→ 外卖优化 + 美团外卖附近推
🎯
缺乏爆品锚点记忆
没有一道让人想起"就是这家"的招牌菜。品牌记忆靠重复一个点，找不到爆品的门店注定是散的。
→ 打造1道视觉+口味双爆品

🔒 私域阵地（留客复购）
👤
扫码即会员 · 零摩擦入会 万能码落座自动弹窗，扫码点餐=入会，新客首单享免费小食。目标：月新增会员80-120人。
💳
储值卡 · 锁客核心武器 3档储值（充100送15 / 充200送35 / 充500送100），POS结账自动弹窗，话术："充200省35，这单直接省XX元"。
🔔
会员触达 · 沉睡唤醒 15天未到店自动触达，发"想你了+专属8折券"，唤醒率目标20%+。月沉睡用户挽回30-50人。
🎯
潜客营销 · 美团独家能力 旗舰版独有：美团平台内的潜客人群定向触达，让附近3公里内有消费意向的用户看到本店优惠推送。
🔗 联动逻辑：公域流量如何转化为私域价值
点评好评 → 自然流量↑ → 新客到店 → 扫码点餐入会 → POS推储值 → 会员复购 → 评价专家引导好评 → 好评再带来新流量。
闭环关键：每一个公域入口都要连接到会员系统，让流量变存量，不白跑一趟。
外卖用户：外卖小卡片引导加微信群，私域维护→堂食转化，外卖不只是出单，是拉新渠道。
🧠
峰终定律 · 顾客体验设计
顾客只记最爽瞬间和走时感受 · 资源集中在峰值+终值两个点
🔥 峰值触点（最爽瞬间）
🍜
出餐仪式感 — 服务员喊桌号上面 端面时服务员统一话术："您的XX面来了！趁热拌开吃更香~"。零成本但创造记忆点。
💥
扫码弹窗惊喜 — 新客专享礼 首次扫码点餐弹出："新客专享！今日免费加卤蛋一枚🥚"。成本约1元，转化率极高。
🎲
小概率惊喜 — 每20单随机1单赠加料 后厨出餐时每20单随机1单多送一份浇头，店员告知"您是今天的幸运客人"。成本<2元，口碑传播力强。
🌟 终值触点（走时感受）
⭐
结账后好评引导 — 自动弹评价页 评价专家：结账后手机自动弹出大众点评好评引导页，无需店员开口，好评率提升3-5倍。
💰
储值结账话术 — 一句话锁客 POS收银自动弹窗，店员话术："您今天消费28元，充100送15元，下次这碗面只要13元！"数字具体、好算清楚。
🍬
收银台小物 — 低成本温度感 收银台放薄荷糖/纸巾自取（成本<0.1元/人）。走时顺手拿一颗，"这家还挺周到"的印象留下了。
